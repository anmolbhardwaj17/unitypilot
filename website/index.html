auto-recover

Most tools hand you code and wish you luck. UnityPilot watches it compile and fixes what breaks.

recovery↺

Hit a compile error? It fixes it itself.

It reads the console, finds the exact file and line, edits the script, and re-runs — until it's clean. You just see "done."

# wrote a script with a typo → compile_failed · "CS0103: 'transfrm' does not exist" · line 6 # reads console, fixes the file, re-runs → recompiled ✓ component attached ✓
