06 SYSTEMS · ALL NOMINAL

Talk to Claude Code like a teammate. UnityPilot works the controls — you just read the results.

INST 01 · SETUP⌖

Sets Unity up for you

Installs the right editor and creates the project. You never open Unity Hub or click around.

INST 02 · SCENE⌖

Builds the scene

Describe a level and it creates scenes, drops in objects, and frames the camera — in a live editor you can watch.

INST 03 · CODE⌖

Writes the game code

It writes C#, compiles it, and wires it to your objects — movement, spinning, jumping, whatever you ask.

INST 04 · ASSETS⌖

Brings your assets in

Drop in models and textures and it imports them into the project, ready to use.

INST 05 · VISION⌖

Shows you the result

Snaps a screenshot of the game view straight into the chat — so the agent (and you) can see it.

INST 06 · TEST⌖

Runs the tests

Kicks off Unity's Test Runner and reports what passed and what failed.

It debugs its own code.

FAIL-SAFE · AUTO-RECOVER

Most tools hand you code and wish you luck. UnityPilot watches it compile and fixes what breaks.

RECOVERY LOG↺

Hit a compile error? It fixes it itself.

It reads the console, finds the exact file and line, edits the script, and re-runs — until it's clean. You just see "done."
