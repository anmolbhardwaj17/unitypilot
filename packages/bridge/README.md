# Bridge (placeholder)

This package will hold the forked C# Unity bridge — a Unity package that runs
*inside* the editor and exposes scene/script/asset/test tools over a WebSocket.

**It is intentionally empty until Phase 4.** Per `SPEC.md §7` and `CLAUDE.md`, the
fork base is chosen by a short spike at the start of Phase 4 (default candidate:
`CoderGamester/mcp-unity`; fallback: `CoplayDev/unity-mcp`). Nothing is vendored or
committed here before that decision — including its LICENSE, which must be verified
against the actual repo file, not its description.

Do not add bridge code in Phases 0–3.
